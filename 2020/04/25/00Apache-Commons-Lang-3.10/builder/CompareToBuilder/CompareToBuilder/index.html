CompareToBuilder

guo_jingtao 2020-04-25 23:26:44

Assists in implementing Comparable.compareTo(Object) methods.

All Methods:

append(boolean[] lhs, boolean[] rhs)

Modifier and Type Method and Description
CompareToBuilder Appends to the builder the deep comparison of two boolean arrays.

toComparison()

Modifier and Type Method and Description
int Returns a negative integer, a positive integer, or zero as the builder has judged the “left-hand” side as less than, greater than, or equal to the “right-hand” side.
  • 多用于自定义对象排序，将多个字段封装为java bean，然后根据多个字段排序
public class MyClass {
   String field1;
   int field2;
   boolean field3;

   ...

   public int compareTo(Object o) {
     MyClass myClass = (MyClass) o;
     return new CompareToBuilder()
      // .appendSuper(super.compareTo(o)
       .append(this.field1, myClass.field1)
       .append(this.field2, myClass.field2)
       .append(this.field3, myClass.field3)
       .toComparison();
   }
 }
